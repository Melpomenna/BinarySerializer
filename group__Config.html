Подсказка компилятору о наиболее вероятном значении условия

Оптимизирует генерацию кода для случаев, когда условие почти всегда истинно. Помогает процессору правильно предсказать переход и уменьшить задержки.

Parameters

Note

// Обычный случай - быстрый путь
return process(ptr);
}
// Редкий случай - медленный путь
return ERROR_NULL_PTR;

◆ BINARYSERIALIZER_NODISCARD

#define BINARYSERIALIZER_NODISCARD   __attribute__((warn_unused_result))

Макрос для предупреждения о неиспользуемом возвращаемом значении

Генерирует предупреждение компилятора, если результат функции игнорируется. Используется для функций, где игнорирование результата может привести к ошибкам (например, функции выделения памяти или проверки ошибок).

Warning
Игнорирование возвращаемого значения может привести к проблемам с дальнейшей логикой или ресурсами
BINARYSERIALIZER_NODISCARD void* allocate(size_t size);
// Компилятор выдаст предупреждение:
allocate(100); // Warning: ignoring return value

Подсказка компилятору о маловероятном значении условия

Оптимизирует генерацию кода для случаев, когда условие почти всегда ложно. Помогает компилятору расположить редко выполняемый код отдельно.

Parameters
__conditionПроверяемое условие
Note
Используется для обработки ошибок и исключительных ситуаций
if (BINARYSERIALIZER_UNLIKELY(file == NULL)) {
// Редкий случай - ошибка открытия файла
return ERROR_FILE_NOT_FOUND;
}
// Обычный путь выполнения
return readFile(file);
